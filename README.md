# Packet-Tracer-Labs
Cisco Packet Tracer Labs
## CCNA Packet Tracer Labs

This repository contains hands-on Cisco Packet Tracer labs
created while studying for the CCNA.

Focus areas:
- Switching (VLANs, trunking, inter-VLAN routing)
- Routing (static, OSPF)
- Network services (DHCP, NAT, ACLs)

All labs were built and tested manually.
